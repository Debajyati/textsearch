# textsearch
basic cli created only for the purpose of learning ocaml

# Installation
You need to have **ocaml(>=5.2)** and **dune** pre-installed.
```bash
git clone https://github.com/Debajyati/textsearch
cd textsearch
dune install
```
# Usage
Run, - 
```bash
textsearch --help
```
to know how to use it
